# Eye Clinic MMC - Homepage Redesign

A modern, wave-inspired homepage redesign for Eye Clinic MMC, an international eye care center specializing in advanced treatments like stem cell therapy, LASIK, cataract surgery, glaucoma management, and pediatric ophthalmology.

## Design Inspiration

This design draws heavy inspiration from the **Cogni:wave mental health website**, adapting its soothing, wave-inspired aesthetic to evoke clarity, trust, and gentle vision restoration. Mental wellness waves are transformed into fluid, light-ray motifs that symbolize improved eyesight and calm healing.

## Getting Started

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Development Server

The app will run on `http://localhost:3000` by default.
